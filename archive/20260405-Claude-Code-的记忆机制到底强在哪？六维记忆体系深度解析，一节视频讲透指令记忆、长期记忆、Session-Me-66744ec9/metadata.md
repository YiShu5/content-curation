## 金句

> "长期记忆严格规定了四种封闭类型，只有这四种信息才值得被记住，其他如代码模式、架构分析、文件路径全都不存，因为它们可以从代码本身推导出来。"

> "如果只记录错误，模型会变得过度保守；记录成功的反馈，是为了告诉 AI 这种做法是对的，以后继续这样。"

> "AutoDream 的触发条件是双重门控：至少距离上次整合过去了24小时，且至少有5个不同的绘画产生了新记忆。"

> "Session Memory 不是等到压缩时才生成摘要，而是从对话过程中就开始持续维护一份绘画笔记。"

> "缓存设计宁可牺牲实时性也要保证确定性，因为如果在模型响应过程中规则突然变了，行为会变得不可预测。"
