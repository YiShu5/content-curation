## 金句

> "记忆是真正区分聊天机器人和 Agent 的东西，因为聊天机器人给出回应，而 Agent 能给出由持久知识和积累经验塑造的回应。"

> "情景记忆是最难做对的类型，因为你要决定删除什么——信息何时过时？如果用户换了工作，旧的项目记忆是保留还是遗忘？"

> "Agent 不会一次性把所有技能加载到工作记忆中，而是只看到一个轻量级索引，当任务匹配时再加载完整指令。"

> "没有语义记忆，Agent 注定会一遍又一遍犯同样的错误，因为它没有可调用的持久知识。"
