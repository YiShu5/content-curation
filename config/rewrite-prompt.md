## 质量标准

1. chinese_title 应体现内容的核心价值，而非字面翻译
2. key_quotes 必须是原文的实际内容，不得编造
3. deep_summary 应保持逻辑连贯，让读者在不看原视频的情况下获得完整认知
4. 如果内容主要是中文，则保留原有措辞风格；如果是英文内容，则用地道中文表达
5. guests 如果无法识别则返回空数组 []
6. 输出必须是合法 JSON，不包含 markdown 代码块标记
